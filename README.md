# 🚀 Social Media & Digital Marketing Platform

A modern **digital marketing platform website** built to help businesses discover and access services such as **website development, PR, and video editing** through an intuitive and conversion-focused interface.

The platform provides a clean, scalable frontend with smooth animations and an AI-powered chatbot to enhance user interaction and automate client queries.

---

## 🌐 Features

- 📈 Multi-service digital marketing platform
- 💬 AI-powered chatbot for automated query handling
- 🎨 Modern UI with responsive design
- ⚡ Smooth animations using Framer Motion
- 📱 Fully responsive across all devices
- 🔧 Reusable and scalable component-based architecture

---

## 🛠 Tech Stack

- **React.js**
- **Tailwind CSS**
- **Framer Motion**
- **TypeScript**

---

## 📊 Project Highlights

- Designed a **conversion-focused user experience** to improve service discovery
- Built **scalable UI components** for easy maintenance and future expansion
- Integrated **interactive chatbot support** for real-time user engagement
- Optimized UI performance for smooth cross-device interaction
